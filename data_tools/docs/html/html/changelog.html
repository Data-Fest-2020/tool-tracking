Version 0.5.1

  • added option for reading all measurement series of a certain tool to MeasurementDataReader.read()

  • fixed occasionally missing labels when reading data

  • refactoring of MeasurementDataReader._read_measurements()

  • enhance test by checking stdout on warnings and errors

  • added two new options to MeasurementDataReader.read():

    • exclude_labels

  • refactoring in data.py and reader.py

  • added a staticmethod MeasurementDataReader.to_ts_data() to convert the read data for usage with seglearn

Version 0.4.3

  • added new caching feature when reading with MeasurementDataReader.read()

Version 0.4.2

  • fixed concatenation option of MeasurementDataReader.read()

  • replaced use of pandas.Dataframe.from_csv() (deprecated) to pandas.read_csv()

  • fixed option to select multiple measurement series in MeasurementDataReader.read()
